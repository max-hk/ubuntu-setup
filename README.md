# ubuntu-setup

## Run

- 
  ```
  chmod +x ./setup.sh
  ./setup.sh
  ```

- Install font Roboto, Noto Sans CJK HK

- Install dash-to-panel

- In gnome-tweaks,

  - Change font to Roboto
  
  - Change Theme to "Yaru Roboto"
  
  - Turn on dash-to panel

- In "Startup Applications",

  - Add Imwheel  
    Command: `imwheel`
  
  - Add Chrome  
    command: `google-chrome --no-startup-window`
